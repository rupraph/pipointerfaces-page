Midi

Prototype

The modules are still under development. Hence these directions are only valid for protytpe testing. Note: some features are not yet implemented, and some are not mature yet.

Any feedback is welcome !

Midi output format can be chosen independently for each sensor axis present on a pipo: for eg one axis can be used for Midi sending notes, while the other can send continuous controls (CC) When notes are chosen, the user can choose which scale or arpegio will be used, as well as the base note and the number of notes spread along the sensing range.

For USB, Pipo is using generic drivers and should be detected righ away by your device.

Note: when connected to a compatible receiving software, HiResolution midi CC can be enabled.
