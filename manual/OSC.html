Configuring OSC first time can be tricky if your are not easy with network configs. Indeed, depending on which network, or security measures on your machine, the data could potentially be blocked. First important thing is to connect through a wifi you know or own, and has a standard config. Like a home wifi, or a wifi you setup yourself. Typically, connecting on public wifi, compagny/work wifi, guests wifi, etc... will likely not work. These usualy have non standard configuration, and complex configurations.

  1. connect your pipo to the wifi, following the wifi manual
  2. make sure the receiving PC is also connected on the same wifi network
  3. identify what is the IP of the destination/receiving machine (the PC)
  4. Go on the Pipo config page and set this destination IP in the OSC section. Note the default port.
  5. make sure the software listening to OSC does it on the same port as the one set in the pipo. usually, 8000 should work.
